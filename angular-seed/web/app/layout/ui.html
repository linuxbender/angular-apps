UI Bausteine

Hier ist eine Auflistung von UI Bausteinen in der App

Beispiel Reportbuilder

Das ganze ist ein Auflistung der möglichen Templates beim erstellen von Report Regeln.

1# String Template

Das ist ein Test

Bitte wählen Sie eine Spalte

String beinhaltet Zahlen, Buchstaben und Sonderzeichen.

z.B hans.muster@mail1974.dev

2# Bool Template

Bitte wählen Sie eine Spalte

Boolen Wert können zwei Zustände haben: True oder Flase

z.B Leistungs-Code ist True

3# Zahlen Template

Bitte wählen Sie eine Spalte

Zahlen können aus Ganzzahlen und Reelenzahlen bestehen.

z.B 1, 2, 3 oder 0.2, 2.25

4# Datum Template

Bitte wählen Sie eine Spalte

Datum Werte verwenden die Formatierung der Landessprache in Kurzform (Tag.Monat.Jahr).

z.B 01.01.1974
